# truck-map

Карта маршрутов для большегрузов (Leaflet + OSM) с объездом весовых рамок и Платона.

## Файлы
- index.html — основная страница карты
- weigh_frames.geojson — точки весового контроля
- platon.geojson — точки Платона

## Публикация
1. Загрузить файлы в GitHub репозиторий (папка root).
2. В Settings → Pages включить публикацию из ветки main.
3. Открыть ссылку вида https://USERNAME.github.io/truck-map/index.html
